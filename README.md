# Bitácora Personal

Este es un programa sencillo en Python para llevar una bitácora personal. Permite:

- Agregar entradas con fecha y hora.
- Ver todas las entradas guardadas.
- Buscar entradas por palabra clave.

## Cómo usarlo

1. Clona o descarga este repositorio.
2. Ejecuta `bitacora.py`
3. Sigue el menú para agregar, ver o buscar entradas.

## Estructura de archivos

- `bitacora.py`: Código fuente del programa.
- `bitacora.txt`: Archivo donde se guardan las entradas.
